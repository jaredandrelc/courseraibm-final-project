# Simple Interest Calculator

A script that calculates simple interest based on user input for principal, rate of interest, and time period.

## Description

This project is a simple Bash script designed to compute Simple Interest using the formula:

$$
Simple \ Interest = \frac{Principal \times Rate \times Time}{100}
$$

This script is intended for educational purposes as part of the "Introduction to Git and GitHub" course.

## Input

The script accepts the following inputs from the user:
* **p**: Principal amount
* **t**: Time period in years
* **r**: Annual rate of interest

## Output

The script outputs the calculated **Simple Interest**.
